Les boucles :

while do → tant que c'est vrai je fais. Pas obligé de rentrer dedans. On test l'expression au départ de la boucle.
Repeat – until → rentre dedans au moins une fois.
For – do → valeur_début TO valeur_fin DO instruction

Tableaux unidimensionnels :

{Exemple}

VAR

Les paramètres

Il existe 2 types :
Les paramètres formels → Ce sont la définition du nombre et du type de valeurs que devra recevoir le sous-programme pour se mettre en route avec succès. Déclaré pendant la déclaration du sous-programme.
Les paramètres effectifs → Ce sont des valeurs réelles (const. ou variables) reçues par le sous-programme au cours de l’exécution du bloc principal. On les définit indépendamment à chaque appel du sous-programme dans l'algorithme principal.

1) Passage paramètre par valeur :
Utiliser la valeur de la variable et à la sortie du sous-programme lui restituer cette valeur malgré les éventuelles modifications subies.
Syntaxe :
PROCEDURE(nom_procédure) (param1 : type1 ; param2, param3: type2)
FONCTION(nom_fonction) (param1 : type1 ; param2, param3 : type2) : Type
2) Passage paramètre par adresse (ou par variable) :
Permet de modifier la nouvelle valeur à la sortie.
FONCTION(nom_fonction) (var param1 : type1 ; var param2, var param3: type2)

Les enregistrements :

Permet de créer une sorte de conteneur qui peut avoir plusieurs attributs/champs. Possibilité d'avoir des types différents !
Ex : Un point est un entier(x, y, z) avec une couleur.
La déclaration d'un objet de type Enregistrement se fait de manière suivante :
TYPE(nom_objet) = Enregistrement
(champ 1) : (type1)
(champ 2) : (type2)
(champ 3) : (type3)
FINENREGISTREMENT
Remarques :
(champ i) → variable à part entière représentant une propriété de l'objet.
(type i) = type de indépendant des autres types de l'objet. Ainsi il peut être simple ou composé, à condition d'être défini avant l'enregistrement.
Manipuler un objet de type enregistrement revient à manipuler chacun des champs qui les constituent. Cela se fait grâce à la variable Nom_objet.nom_champ.
Ex : voitures.garantie ← vrai ou encore Lire(voitures.puissance)
NB : Nom_objet.nom-champ.nom_sous_champ
